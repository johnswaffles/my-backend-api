# Tiny Hero Quest (Godot)

Godot project for the Tiny Hero Quest game.

## Purpose
- Core gameplay logic and scenes for the tiny-hero-quest project.
- Current direction: a compact dungeon-crawler RPG prototype, starting with Dungeon Level 1.

## Build/Export
- Export for web and publish to the web host folder used by your deployment flow.

## Notes
- Separate from Cozy Builder.
- Keep this folder focused on RPG gameplay and assets.
- The first dungeon floor is procedurally assembled in `scripts/main.gd` from simple Godot primitives.
